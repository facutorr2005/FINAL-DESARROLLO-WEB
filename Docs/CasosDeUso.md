# DEFINICIÓN DE CASOS DE USO DE OUTLET VIRTUAL

## ACTORES:
* **Usuario** (Persona o Empresa)
* **Vendedor**
* **Personal de Secretaría**
* **Administrador**

---

## 1. Actor: Usuario (Persona o Empresa)

* Como **Usuario**, quiero poder registrarme eligiendo entre perfil Persona o Empresa, para que el sistema me solicite los datos impositivos correctos (CUIL o CUIT).
* Como **Usuario**, quiero iniciar sesión con mi CUIL/CUIT y contraseña, para acceder de forma segura a mi panel personal.
* Como **Usuario**, quiero poder solicitar la recuperación de mi clave a través de mi CUIT/CUIL, mediante un correo electrónico, para volver a ingresar en caso de olvido.
* Como **Usuario**, quiero poder modificar mis datos o darme de baja, para tener control total sobre mi información personal y mi estado en el sistema.
* Como **Usuario**, quiero navegar por categorías y ofertas, para encontrar rápidamente los productos de outlet que me interesan.
* Como **Usuario**, quiero poder agregar productos a una lista actual, para visualizar el detalle y el costo estimado de los artículos antes de confirmar.
* Como **Usuario**, quiero cerrar y enviar mi lista, para que el negocio reciba mi interés y se ponga en contacto conmigo para concretar la venta.
* Como **Usuario**, quiero visualizar el historial de mis listas, para hacer un seguimiento de mis pedidos anteriores y ver en qué estado se encuentran.

---

## 2. Actor: Vendedor (Gestión de Ventas)

* Como **Vendedor**, quiero ingresar al panel de gestión con mis credenciales asignadas, para acceder a la administración de pedidos.
* Como **Vendedor**, quiero poder modificar el estado de las listas (Pendiente, En Proceso, Finalizada), para mantener organizada la atención a los clientes.
* Como **Vendedor**, quiero acceder al detalle de cada lista, para conocer los productos solicitados y los datos de contacto (CUIL, Teléfono) del cliente.

---

## 3. Actor: Personal de Secretaría (Gestión de Catálogo)

* Como **Personal de Secretaría**, quiero ingresar al panel de gestión con mi usuario, para administrar la mercadería del sitio.
* Como **Personal de Secretaría**, quiero poder cargar nuevos productos, para mantener actualizado el catálogo de outlet con sus especificaciones técnicas y fotos.
* Como **Personal de Secretaría**, quiero gestionar el stock, precios y ofertas de los productos existentes, para que la información que ve el cliente sea veraz y actual.

---

## 4. Actor: Administrador (Administrador)

* Como **Administrador**, quiero ingresar al panel de control total, para supervisar el funcionamiento integral del sistema.
* Como **Administrador**, quiero poder crear, editar o dar de baja cuentas de Staff (Vendedores y Personal de Secretaría), para gestionar el equipo de trabajo del local.
